# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

Augentic AI marketing website — a static site for an AI systems integrator. Hosted on Cloudflare Pages at **augenticai.com**. GitHub repo: `cora-catchadvisors/augentic-ai`.

## Architecture

This is a **zero-dependency static site** — no build tools, no frameworks, no package.json. All pages are self-contained HTML files with embedded CSS and JS.

### Page structure

- `index.html` — Main homepage (single-page with anchor sections: hero, problem, solution, offerings, process, who-it-is-for, authority, contact)
- `blog/index.html` — Blog listing page
- `blog/<slug>/index.html` — Individual blog posts (10 posts)
- `book/index.html` — Strategy call booking page
- `guide/index.html` — AI Readiness Guide (interactive assessment with checkbox scoring)
- `privacy/index.html` — Privacy policy
- `terms/index.html` — Terms of service
- `404.html` — Custom 404 page

### Build script

`_build_site.py` is a Python script that was used to generate the blog posts, blog index, booking page, and guide page. It contains shared `NAV`, `FOOTER`, `BASE_CSS`, and `BASE_JS` template strings, plus blog post content defined in a `POSTS` list. Note: the output paths are hardcoded to `/Users/cora/Projects/augentic-ai/` — update these if running it again.

The homepage (`index.html`) was **not** generated by the build script — it is hand-crafted and standalone.

### Shared conventions across all pages

- **Fonts**: Google Fonts — Inter (body) + Playfair Display (headings)
- **Color palette**: `--bg: #0A0A0A`, `--text: #F5F5F0`, `--accent: #D4AF37` (gold), `--bg-card: #111111`
- **Light/dark theme**: Toggle via `body.light` class, persisted in `localStorage('theme')`
- **Cookie consent banner**: Appended to all pages, state in `localStorage('augentic_cookie_consent')`
- **Navigation**: Sticky nav with scroll detection, hamburger menu on mobile (<900px)
- **Animations**: Intersection Observer fade-in on scroll (homepage)
- **Inline SVG logo**: The "A" logomark and "AUGENTIC AI" wordmark are inline SVGs in both nav and footer

## Development

No dev server — open HTML files directly in a browser or use any static file server:

```bash
python3 -m http.server 8000    # then visit localhost:8000
npx serve .                     # alternative
```

To regenerate blog/book/guide pages from the build script (after fixing paths):

```bash
python3 _build_site.py
```

## Deployment

Auto-deploys to Cloudflare Pages on push to `main`. No build command configured — Cloudflare serves the files directly.

## Editing guidelines

- Each page is fully self-contained (CSS + JS embedded). Changes to shared elements (nav, footer, theme toggle, cookie banner) must be replicated across all pages manually or by updating `_build_site.py` and regenerating.
- The `index.html` homepage is standalone and not managed by the build script.
- `robots.txt` disallows `build-prompt.md` and `_build_site.py` from indexing.
- `build-prompt.md` contains the original design spec/prompt used to generate the site — useful as a reference for tone, structure, and design intent.

## Design tone

Executive, confident, direct, outcome-driven. No hype, no buzzwords. Premium consulting firm aesthetic. See `build-prompt.md` for the full design specification.
